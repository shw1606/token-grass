# 개인정보 처리방침

_최종 업데이트: 2026-07-05_ · [English](PRIVACY.md)

**TokenGrass는 아무것도 수집하지 않습니다.** 백엔드도, 계정도, 분석 도구도 없습니다.
앱이나 Mac 앱에서 하는 어떤 것도 저희나 저희가 관리하는 제3자에게 전송되지 않습니다.
애초에 데이터를 받을 "저희" 서버 자체가 없기 때문입니다.

## 앱이 다루는 것, 그리고 그게 머무는 곳

TokenGrass는 여러분의 Claude Code 사용량을 잔디 그래프("잔디") 위젯으로 보여줍니다.
모든 처리는 여러분의 기기와 Apple iCloud 안에서만 일어납니다.

- **Mac에서**, 무료 TokenGrass 앱이 기존 Claude Code 사용량을 읽습니다. Claude Code
  로그인은 macOS 키체인에서 읽어 Anthropic으로부터 사용량을 가져오는 데에만 쓰입니다.
  토큰은 Mac을 떠나지 않으며, TokenGrass는 토큰을 저장하지 않습니다.
- Mac 앱은 작은 일별 "사용 강도" 요약(수 킬로바이트, 날짜와 퍼센트뿐이며 **토큰 개수도,
  대화 내용도, 자격 증명도 없음**)을 만들어 여러분의 개인 **iCloud 키-값 저장소**에 씁니다.
- **아이폰에서**, 앱과 위젯이 그 요약을 iCloud에서 읽어 잔디를 그립니다. 이게 전부입니다.

저희는 이 데이터를 받거나, 저장하거나, 접근하지 않습니다. iCloud 동기화는 여러분의
Apple ID와 [iCloud 개인정보 약관](https://www.apple.com/legal/privacy/) 아래에서
Apple이 처리합니다.

## 제3자

- **Anthropic**: Mac 앱이 여러분 자신의 Claude 자격 증명으로 Anthropic 사용량
  엔드포인트를 직접 호출해 여러분 자신의 사용량을 읽습니다. 이는 여러분과 Anthropic
  사이의 일이며(Anthropic 개인정보 처리방침 참고), TokenGrass는 독립적인 프로젝트로
  Anthropic과 제휴 관계가 없습니다.
- **Apple iCloud**: 작은 사용량 요약을 여러분의 기기 사이에서 동기화하는 데에만
  사용됩니다.

광고, 추적, 제3자 SDK, 크래시 리포팅 없음.

## 데이터 삭제

모두 지우려면 아이폰에서 앱을 삭제하고 Mac 앱을 종료하거나 제거하세요. 동기화된 요약까지
지우려면 기기 설정에서 TokenGrass의 iCloud 데이터를 삭제하면 됩니다. 어떤 서버에도
저장된 것이 없으므로 그 밖에 지울 것은 없습니다.
